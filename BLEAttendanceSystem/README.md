 # BLE Attendance System using ESP32

## Introduction

This project is a BLE Attendance System developed using ESP32 and Linux terminal tools.

The system scans nearby BLE devices and marks attendance automatically based on registered MAC addresses.

This project was developed as part of embedded systems and IoT learning using Arduino CLI, Linux terminal workflow, Git, and GitHub.
