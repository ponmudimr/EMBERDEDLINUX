# BLE Attendance System using ESP32

This project uses ESP32 BLE scanning to detect registered devices and mark attendance automatically.

## Features

- BLE device scanning
- Attendance detection
- ESP32 based system
- Linux terminal workflow
- Arduino CLI support

## Hardware

- ESP32
- Mobile devices
- WiFi (optional)

## Future Improvements

- Web dashboard
- Google Sheets integration
- OLED display
- Database logging
